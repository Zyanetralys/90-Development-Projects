# Metodología de Evaluación

## ¿En qué se basa este test?

Este assessment evalúa preferencias y tendencias en dinámicas BDSM mediante preguntas estructuradas. Es una herramienta de autoconocimiento.

## Estructura del Test

### 120 Ítems Totales
- **100 ítems clínicos** - Miden las 10 dimensiones principales
- **20 ítems de validez** - Detectan respuestas inconsistentes

### Las 10 Dimensiones

| Dimensión | Qué mide | Ítems |
|-----------|----------|-------|
| Power Exchange | Dominancia vs Sumisión | 20 |
| Sensation | Tolerancia al dolor/intensidad | 20 |
| Primal | Instintos animales (caza/presa) | 15 |
| Nurturing | Caregiver vs Little | 15 |
| Pet Play | Identificación con rol animal | 15 |
| Ownership | Propiedad vs Ser propiedad | 15 |
| Bondage | Preferencia por ataduras | 10 |
| Observation | Voyeurismo vs Exhibicionismo | 10 |
| Service | Orientación al servicio | 10 |
| Switch | Flexibilidad de rol | 10 |

## Sistema de Puntuación

### Escala Likert (1-5)

1 = Totalmente en desacuerdo
2 = En desacuerdo
3 = Neutral
4 = De acuerdo
5 = Totalmente de acuerdo


### Puntuaciones
Los resultados se comparan con rangos de referencia para identificar tendencias significativas.

**Interpretación:**
- < 40 = Bajo
- 40-60 = Promedio
- > 60 = Alto

## Patrones de Código

El sistema identifica 2-3 tendencias principales y genera un código que describe tu combinación de rasgos.

## Limitaciones

- No valida consentimiento real
- No evalúa salud mental
- Resultados dependen de honestidad del usuario

# Escalas de Validez

## ¿Por qué existen?

Este assessment incluye ítems diseñados para detectar cuando alguien no está respondiendo con honestidad o consistencia. Si las escalas de validez fallan, los resultados pueden no ser precisos.

## Las 5 Escalas

### 1. Escala L - Deseabilidad Social
**8 ítems** que detectan si estás tratando de parecer "demasiado bueno"

Ejemplo: *"Nunca he dicho una mentira, ni siquiera pequeña"*

**Cutoff:** Puntuación alta = Perfil posiblemente inflado

### 2. Escala F - Respuestas Raras
**6 ítems** con afirmaciones que casi nadie experimenta

Ejemplo: *"Puedo controlar el clima con mis pensamientos"*

**Cutoff:** Puntuación muy alta = Posible respuesta aleatoria

### 3. Escala K - Defensividad
Mide qué tanto te estás "cubriendo" psicológicamente

**Cutoff:** Puntuación alta = Posible defensividad

### 4. VRIN - Inconsistencia Variable
**3 pares de ítems** similares que deberían responderse igual

**Cutoff:** Puntuación alta = Respuestas inconsistentes

### 5. TRIN - Sesgo de Respuesta
Mide tendencia a responder siempre "sí" o siempre "no"

**Cutoff:** Puntuación extrema = Posible sesgo

## Niveles de Validez

| Nivel | Criterio | Qué hacer |
|-------|----------|-----------|
| **VÁLIDO** | 0 flags | Interpretar normalmente |
| **CUESTIONABLE** | 1 flag | Interpretar con cautela |
| **INVÁLIDO** | 2+ flags | Repetir evaluación |
| **ERROR** | Problema técnico | Reportar |

## Consejos para Respondir

1. **Sé honesto** - No hay respuestas "correctas"
2. **No pienses demasiado** - Tu primera intuición suele ser la más precisa
3. **No hay juicio** - Esto es para ti
4. **Si no aplica** - Elige "Neutral"
5. **Tómate tu tiempo** - 25-35 minutos es lo normal
