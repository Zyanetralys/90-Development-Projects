==================================================
VISIÓN GENERAL
==============

Aplicación local multiplataforma para gestionar recetas como conocimiento estructurado.
Incluye búsqueda avanzada por ingredientes, etiquetas, notas, favoritos, variantes,
exportación profesional, imágenes y motor de decisión culinaria.

==================================================
FUNCIONALIDADES PRINCIPALES
===========================

* CRUD completo de recetas
* Ingredientes normalizados con alias
* Búsqueda por ingredientes (estricto / contiene / flexible)
* Etiquetas y categorías libres
* Filtro y ordenación avanzada
* Sistema de notas (1–10)
* Favoritos
* Variantes de recetas
* Notas por receta
* Imágenes asociadas
* Exportación a PDF / CSV / Excel / JSON
* Persistencia local robusta

==================================================
MODELO DE DATOS (SQLITE)
========================

## TABLE recipes

id INTEGER PRIMARY KEY
nombre TEXT NOT NULL
descripcion TEXT
pasos TEXT
tiempo INTEGER
dificultad TEXT
cocina TEXT
favorita BOOLEAN DEFAULT 0
rating_actual INTEGER
created_at TEXT
updated_at TEXT

---

## TABLE ingredients

id INTEGER PRIMARY KEY
nombre TEXT UNIQUE
categoria TEXT
alias TEXT

---

## TABLE recipe_ingredients

recipe_id INTEGER
ingredient_id INTEGER
obligatorio BOOLEAN
cantidad TEXT
unidad TEXT

---

## TABLE tags

id INTEGER PRIMARY KEY
nombre TEXT UNIQUE
tipo TEXT

---

## TABLE recipe_tags

recipe_id INTEGER
tag_id INTEGER

---

## TABLE ratings

recipe_id INTEGER
score INTEGER
comentario TEXT
fecha TEXT

---

## TABLE notes

id INTEGER PRIMARY KEY
recipe_id INTEGER
texto TEXT
created_at TEXT

---

## TABLE recipe_variants

id INTEGER PRIMARY KEY
recipe_id INTEGER
nombre TEXT
cambios TEXT

---

## TABLE images

id INTEGER PRIMARY KEY
recipe_id INTEGER
path TEXT
descripcion TEXT

==================================================
BÚSQUEDA POR INGREDIENTES
=========================

MODOS:

* ESTRICTO: solo ingredientes seleccionados
* CONTIENE: incluye los seleccionados
* FLEXIBLE: scoring por coincidencia

SCORING BASE:

* +100 si todos los obligatorios están
* +10 por ingrediente presente
* -20 por ingrediente obligatorio faltante
